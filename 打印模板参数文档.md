# 概述

本文档用于说明开燃云集燃气云平台发票模板中可用的参数，客户可参考该文档自行设计发票模板．

本系统使用锐浪报表实现发票打印功能，发票数据通过XML文本传递给模板，其格式如下:

```xml
<xml><Master>{master}</Master>{detail}</xml>
```

`{master}`和`{detail}`部分传递的数据将在下节详细介绍．为了方便描述，接下来我们将使用JSON格式对数据进行说明（本文使用`//`表示注释），如:

```xml
<country>
  <name>中国</name>
  <province>
    <name>黑龙江</name>
    <citys>
      <city>哈尔滨</city>
      <city>大庆</city>
    </citys>  　　
  </province>
</country>
```

我们将使用如下格式进行说明:

```js
{
  "country":{
    "name": "中国",　//国家名
    "provinces": [
        {"name": "黑龙江", "citys": {"city": ["哈尔滨", "大庆"]}},
    ]
  }
}
```

# 模板说明

## 工程费发票模板

### Master

```js
{
  // 开票日期
  "invoice_date": "2018-01-01 10:23:56",
  // 公司名
  "company_name": "XXX天然气公司",
  // 发票号（8位整数）
  "invoice_id": "00000001",
  // 工程ID
  "project_id": "1",
  // 交易流水号
  "translog_id": "12345678",
  // 工程名
  "project_name": "XXXXX",
  // 本次交易金额
  "payment": "88.888",
  // 补打标志，取值范围:（首次，补打）
  "flag": "首次",
  // 收费员姓名
  "operator_name": "张三",
  // 收费员ID
  "operator_id": "999",
  // 收费日期
  "charge_date": "2018-01-01 10:23:56",
}
```

### Detail

```js
// 工程费用详情列表
[
  {
    // 项目
    "name": "开户费",
    // 单位（有的费用无单位）
    "unit": "户",
    // 单价
    "price": "6000",
    // 数量
    "quantity": "1",
    // 金额
    "amount": "6000"
  }
]
```
